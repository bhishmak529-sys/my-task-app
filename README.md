# 🚀 TaskPro Elite 

A full-stack, enterprise-level Task Management Web Application built with Python and Flask. This project features a Kanban-style board, real-time analytics, and automated background tasks.

🌍 **Live Demo:** [Click Here to View App](https://my-task-app-hens.onrender.com)

## ✨ Key Features
* **🔒 Google OAuth Integration:** Secure login system using Google accounts.
* **📊 Productivity Dashboard:** Interactive charts (Doughnut & Bar) using `Chart.js` to track task status and priorities.
* **📋 Kanban Board:** Drag & drop tasks between "Backlog", "To Do", "In Progress", and "Done".
* **✅ AJAX Sub-tasks:** Create checklist sub-tasks without refreshing the page.
* **📎 File Attachments:** Upload files and view images directly in a built-in Lightbox.
* **⏰ Automated Cron Jobs:** Uses `APScheduler` to send daily summary emails and track overdue tasks automatically.
* **🌓 UI/UX:** Fully responsive design with Dark/Light mode toggle and Confetti animations for completed tasks.

## 💻 Tech Stack
* **Backend:** Python, Flask, SQLAlchemy, Authlib
* **Frontend:** HTML5, CSS3, JavaScript, Chart.js
* **Database:** SQLite
* **Deployment:** Gunicorn, Render (Cloud Hosting)
